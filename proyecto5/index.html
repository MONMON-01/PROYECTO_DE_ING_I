PRACTICA 5. CONTRAMOLDE DE CERA

Montserrat Vázquez Guzmán
Ing. Mecatrónica
Fecha: Octubre 10, 2024
Maquinas Mecatronica

- Resumen -

- Aprender la diferencia entre un molde y un contramolde.

- Primer uso de la fresadora en foamular de 1pulg. y cera para crear un contramolde.

- Introducción -

Un contramolde es una forma negativa que replica la forma anatómica de una impresión. Es un paso necesario para la elaboración de cualquier tipo de molde o carcasa.

En este caso la plicaremos en cera para la creacion de un posterior molde de silicon.

- Materiales -

- Software para cargar los archivos

- Bloque de cera

- Archivo stl para el modelo

- Router

- broca(fresa) de 1/4

- broca(fresa) de 1/8

- Desarrollo -

A partir del modelo sacado de thingiverse(platafroma de descarga gratuita de modelos stl para impresion 3D) sacamos un modelo de la figura que deseamos plasmar en la cera.

En este caso utilizamos el modelo de la cabeza de un personaje del vieojuego "Hollow Knight"

Una vez descargado el modelo se prepara en el software para pasarse a una USB como archivo de desvaste y acabado

Es imprtante recordar que el router y las brocas son herramientas delicadas brindadas por la universidad por lo tanto es necesario tomar las precauviones necesarias

Una vez cargados los archivos a nuestra memoria se insertan en el router y los seleccionamos para su preparacion

* El desvaste de la pieza fue realizado con la broca de 1/4

* Mientras que el acabado fue realizado con la broca de 1/8 para acabados mas detallados

- Resultados -

La pieza despues de una amplia seleccion de modelos e intentos por completarla correctamente quedo con un buen tamaño y acabado que nos sirvieron para la siguiente practica
