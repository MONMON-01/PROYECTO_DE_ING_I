PRACTICA 7. FIGURA DE RESINA

Montserrat Vázquez Guzmán
Ing. Mecatrónica
Fecha: Septiembre 16, 2024
Maquinas Mecatronica

- Resumen -

Elaborar una figura de resina utilizando como base el molde de silicona creado en la práctica anterior.

...

- Materiales -

- Molde de silicona

- Resina y sus derivados (proporcionado por el docente)

- Vaso de papel

- Palo de madera

- Bascula

- Desarrollo -

Se utilizó el molde de silicona previamente elaborado como base para la creación de una figura de resina. Siguiendo un procedimiento similar al de la práctica anterior, se preparó la mezcla de resina y catalizador en vasos de papel, en lugar de plástico, para evitar interacciones químicas indeseadas.

La mezcla se realizó cuidadosamente, utilizando la balanza para medir con precisión las proporciones indicadas por el docente. Se empleó el palo de madera para mezclar ambos componentes, asegurando una integración homogénea. Luego, la mezcla se vertió en el molde de silicona con movimientos controlados para evitar burbujas de aire, y se dejó reposar para que endureciera según el tiempo estimado.

- Resultados -

El resultado fue una figura con un acabado visual atractivo y un color homogéneo, lo que reflejó un buen manejo del material y el molde. Sin embargo, durante el proceso se observó que la resina se endureció rápidamente, lo que podría atribuirse a un exceso de catalizador o resina en la mezcla. Este endurecimiento prematuro dificultó una distribución adecuada del material en ciertas áreas del molde, afectando ligeramente el detalle de la figura.
